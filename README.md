# BMI Calculator

A simple, responsive BMI (Body Mass Index) calculator built with HTML, CSS, and JavaScript.

## 🌟 Features

- Calculate BMI using height (cm) and weight (kg)
- Color-coded BMI categories for easy interpretation
- Responsive design that works on all devices
- Clean and modern user interface
- Reset functionality to clear inputs

## 📱 BMI Categories

- **Severely underweight**: BMI < 16.0
- **Underweight**: BMI 16.0–18.4
- **Normal**: BMI 18.5–24.9
- **Overweight**: BMI 25.0–29.9
- **Moderately obese**: BMI 30.0–34.9
- **Severely obese**: BMI 35.0–39.9
- **Morbidly obese**: BMI ≥ 40.0

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

